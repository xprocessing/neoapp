# 用户收货地址管理

## 新增文件

| 文件 | 说明 |
|------|------|
| `dto/UserAddressVO.java` | 地址展示 VO（id/name/phone/province/city/district/detail/isDefault/createTime） |
| `dto/UserAddressRequest.java` | 地址创建/更新请求（含 @NotBlank 校验） |
| `controller/UserAddressController.java` | 地址 CRUD 端点 `/api/user/addresses` |

## API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/user/addresses` | 获取当前用户地址列表（默认在前） |
| POST | `/api/user/addresses` | 新增地址（首个自动设默认） |
| PUT | `/api/user/addresses/{id}` | 更新地址 |
| DELETE | `/api/user/addresses/{id}` | 删除地址（逻辑删除） |
| PUT | `/api/user/addresses/{id}/default` | 设为默认（自动取消其他默认） |

## 前端修改

`web/src/views/Profile.vue` — 新增"收货地址"卡片区域：
- 地址列表展示（姓名/电话/地址全称/默认标签）
- 新增/编辑抽屉（省市区三栏 + 详细地址 + 默认开关）
- 设为默认、编辑、删除操作
- 空状态提示（MapPin 图标）

## 测试验证
- ✅ GET 空列表 → []
- ✅ POST 创建（首个自动默认）→ 200
- ✅ POST 创建第二个（isDefault=0）→ 200
- ✅ PUT /{id}/default 切换默认 → 旧地址取消默认
- ✅ DELETE → 逻辑删除
