# 管理后台区分管理员管理与用户管理

## 目标
管理后台侧边栏拆分"管理员管理"和"用户管理"，分别管理 `admin_user` 表和 `user_info` 表。

## 后端新增

| 文件 | 说明 |
|------|------|
| `entity/AdminUser.java` | admin_user 表实体，含 username/password/nickname/email/phone/avatar/gender/status 等字段 |
| `mapper/AdminUserMapper.java` | MyBatis-Plus BaseMapper |
| `service/AdminUserService.java` | IService 接口 |
| `service/impl/AdminUserServiceImpl.java` | Service 实现 |
| `dto/AdminUserVO.java` | 管理员展示 VO（不含 password） |
| `dto/AdminUserPageQuery.java` | 分页查询参数 |
| `dto/AdminUserRequest.java` | 创建/更新请求 |
| `controller/AdminManageController.java` | `/api/admin/admins` CRUD 端点 |

### API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/admin/admins` | 分页查询管理员列表（支持 keyword/status 筛选） |
| GET | `/api/admin/admins/{id}` | 查询单个管理员详情 |
| POST | `/api/admin/admins` | 新增管理员（密码 BCrypt 加密，用户名唯一校验） |
| PUT | `/api/admin/admins/{id}` | 更新管理员（密码选填，不传则不修改） |
| DELETE | `/api/admin/admins/{id}` | 删除管理员（禁止删除 admin 超级管理员） |

### 安全规则
- 所有接口受 `/api/admin/**` JWT 认证保护
- 不允许删除 username=admin 的超级管理员（返回 403）
- 密码创建时必填，更新时选填
- 用户名唯一性校验

## 侧边栏结构
```
仪表盘
管理员管理  ← 管理 admin_user 表
用户管理    ← 管理 user_info 表
系统设置
```
