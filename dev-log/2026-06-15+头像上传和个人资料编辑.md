# 头像上传和个人资料编辑功能

## 修改时间
2026-06-15 00:40

## 修改内容

### 后端新增/修改

#### 1. COS SDK 集成 (`backend/pom.xml`)
- 添加 `com.qcloud:cos_api:5.6.227` 依赖

#### 2. COS 配置 (`backend/src/main/resources/application-dev.yml`)
- 添加 COS 参数配置（secret-id, secret-key, region, bucket）
- 添加 multipart 上传限制（max 5MB）

#### 3. CosProperties 配置类 (`backend/src/main/java/com/neoapp/config/CosProperties.java`)
- 新建类，读取 `cos.*` 配置
- 字段：secretId, secretKey, region, bucket

#### 4. CosService 服务 (`backend/src/main/java/com/neoapp/service/CosService.java`)
- 封装 COSClient 初始化/销毁
- `uploadAvatar(MultipartFile, userId)` 方法：
  - 校验图片类型（JPG/PNG/GIF/WebP）
  - 校验文件大小（≤5MB）
  - 上传到 COS 路径：`avatar/{userId}/{uuid}.{ext}`
  - 返回完整访问 URL

#### 5. 用户自助更新接口
- 新建 `UserProfileRequest.java` DTO（nickname, sex, avatar）
- `UserService.updateProfile(userId, request)` — 用户自行更新昵称/性别/头像
- `UserController` 新增：
  - `PUT /api/user/profile` — 更新个人资料
  - `POST /api/user/upload/avatar` — 上传头像到 COS

### 前端修改

#### 1. `web/src/utils/request.js`
- 移除默认的 `Content-Type: application/json`，让 axios 自动检测
  - JSON 请求自动设 application/json
  - FormData 请求自动设 multipart/form-data

#### 2. `web/src/views/Profile.vue` — 完整重写
- **查看模式**：头部区域展示头像+昵称+状态标签，详情列表展示手机号/性别/邀请码/注册时间
- **编辑模式**：
  - 头像点击上传，悬停显示相机图标，上传中显示 loading 覆盖
  - t-form 表单编辑昵称（max 50 字符）和性别（未知/男/女）
  - 保存/取消按钮
- 头像上传流程：选择文件 → 前端校验格式和大小 → POST /api/user/upload/avatar → 获取 URL → 更新预览
- 保存流程：表单校验 → PUT /api/user/profile → 刷新用户信息 → 切回查看模式
