### 备注
- 所有 `/api/admin/**` 路径受 JWT 保护，无 Token 或假 Token 均返回 401
- C 端用户 Token 目前也能访问 admin 接口（后端未做角色隔离），但前端路由守卫已阻止非管理员访问 admin 页面
- 超级管理员（username=admin）不可删除（403 保护）
- 逻辑删除使用 MyBatis-Plus @TableLogic，删除后 `is_deleted=1` 但从查询结果中自动隐藏
