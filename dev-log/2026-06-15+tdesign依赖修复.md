# 2026-06-15 tdesign-vue-next 依赖修复

## 问题

Web 端启动报错：
```
Failed to resolve entry for package "tdesign-vue-next". 
The package may have incorrect main/module/exports specified in its package.json.
```

## 原因

`tdesign-vue-next` 安装不完整，`es/index.mjs` 文件缺失，导致 Vite 无法解析 ESM 入口。

## 修复

- 重新安装 `tdesign-vue-next`：`npm install tdesign-vue-next@latest --save`
- 升级到 1.20.1 版本
- 重启 web 开发服务器
