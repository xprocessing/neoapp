## 二、后端 (backend)

### 技术栈
- Java 21 + Spring Boot 3.4.1
- MyBatis-Plus 3.5.9
- Spring Security 6.x + JWT
- MySQL 8.0 + HikariCP
- Maven 构建

### 目录结构
```
backend/
├── pom.xml
└── src/main/
    ├── java/com/neoapp/
    │   ├── NeoAppApplication.java          # 启动类
    │   ├── common/
    │   │   ├── Result.java                 # 统一响应封装 {code, msg, data}
    │   │   ├── BusinessException.java      # 业务异常
    │   │   └── GlobalExceptionHandler.java # 全局异常处理器
    │   ├── config/
    │   │   ├── SecurityConfig.java         # Spring Security 安全配置
    │   │   ├── MyBatisPlusConfig.java      # MyBatis-Plus 配置(分页/填充/逻辑删除)
    │   │   ├── CorsConfig.java             # CORS 跨域配置
    │   │   └── WebMvcConfig.java           # Web MVC 配置
    │   ├── controller/
    │   │   └── HealthController.java       # 健康检查 /api/health
    │   ├── service/                        # 业务逻辑层 (预留)
    │   ├── mapper/                         # Mapper接口 (预留)
    │   ├── entity/
    │   │   └── BaseEntity.java             # 实体基类 (id, createTime, updateTime, isDeleted)
    │   └── dto/                            # 数据传输对象 (预留)
    └── resources/
        ├── application.yml                 # 公共配置
        ├── application-dev.yml             # 开发环境配置 (MySQL连接)
        └── db/
            └── schema.sql                  # 初始建表 SQL
```

### 关键配置说明
- **统一响应格式**：`Result<T>` 类，code=200 成功，其他失败
- **安全配置**：开发模式放行 `/api/health/**`, `/api/auth/**`, `/api/public/**`，其他接口需认证；401/403 返回 JSON
- **CORS**：允许所有来源，支持 GET/POST/PUT/DELETE/PATCH/OPTIONS
- **端口**：8080
- **数据库**：`jdbc:mysql://localhost:3306/neoapp`，用户名/密码 root/root
- **初始数据**：默认管理员 admin/admin123 (BCrypt 加密)
- **逻辑删除**：字段 `is_deleted`，0 未删除 / 1 已删除

### 启动方式
```bash
cd backend
mvn spring-boot:run
```

## 三、管理端 (admin)

### 技术栈
- Vue 3.5 + Composition API
- Vite 5
- TDesign Vue Next 1.13
- Tailwind CSS 3.4
- Vue Router 4 + Pinia 2
- Axios (请求/响应拦截)

### 目录结构
```
admin/
├── package.json
├── vite.config.js
├── tailwind.config.js
├── postcss.config.js
├── index.html
└── src/
    ├── main.js                 # 入口
    ├── App.vue                 # 根组件
    ├── router/index.js         # 路由 (/login, /dashboard)
    ├── utils/request.js        # Axios 封装 (Token注入, 统一错误)
    ├── stores/user.js          # Pinia 用户状态
    ├── styles/global.css       # 全局样式 + 响应式断点 + CSS变量
    ├── layouts/
    │   └── AdminLayout.vue     # 侧边栏 + 顶栏布局 (移动端抽屉适配)
    └── views/
        ├── Login.vue           # 登录页 (品牌左 + 表单右, 移动端上下布局)
        ├── Dashboard.vue       # 仪表盘 (统计卡片 + 预留区)
        └── NotFound.vue        # 404 页面
```

### 关键配置说明
- **端口**：3000
- **代理**：`/api` → `http://localhost:8080`
- **响应式断点**：<768px 手机 / 768-1200px 平板 / >1200px PC
- **侧边栏**：PC 端 240px 可折叠至 64px；移动端固定抽屉
- **登录**：模拟登录逻辑，存 token 到 localStorage
- **主题色**：`#0052D9` (TDesign 品牌蓝)

### 启动方式
```bash
cd admin
npm install
npm run dev
```
